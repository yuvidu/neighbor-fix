NeighborFix or PawuraWatch (පවුර වැඩිපුර in Sinhala) work well—short, memorable names highlighting local problem-solving without overlapping existing apps like ManKiwwa. "NeighborFix" emphasizes community action while "PawuraWatch" adds authentic Sri Lankan flavor for Negombo users.
​

App Summary
NeighborFix is a web app empowering Sri Lankan communities to report and resolve hyper-local issues like garbage piles, broken streetlights, noisy neighbors, flooded roads, and illegal dumping. Users snap photos, pin GPS locations, and describe problems in Sinhala/English/Tamil—creating a live map where neighbors vote on urgency and track fixes in real-time.

Key User Flows
Report: Photo + GPS + description → instantly appears on community map

Vote/Discuss: Neighbors upvote critical issues, chat via your existing stranger app integration

Track: Status updates (Reported → Assigned → Fixed) with before/after photos

Dashboard: Pradeshiya Sabha view for officials to claim/resolve reports

Technical Overview
Built with React/MUI (your LayoutMain pattern), Leaflet maps, Firebase backend. MVP targets Western Province (Negombo/Colombo) with offline-first photo uploads. Launch strategy: WhatsApp sharing + local council partnerships for official adoption.
